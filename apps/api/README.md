# revWave API

NestJS backend for revWave reputation management platform.

## Development

```bash
# Start in watch mode
pnpm dev

# Build
pnpm build

# Start production
pnpm start:prod
```

## Environment

Copy `.env.example` to `.env` and configure all required variables.

## Health Check

Once running, visit http://localhost:3001/health to verify the API is working.
